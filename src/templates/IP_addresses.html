An informative page about IP Addresses

Every device connected to the internet is assigned a unique IP address. An IP address (Internet Protocol address) is a unique string of numbers separated by periods (IPv4) or colons (IPv6) that identifies each device connected to a network. It's like a digital address that allows devices to communicate with each other over the internet, ensuring data is sent and received accurately.

The two main protocols for IP addresses currently are IPv4 and IPv6. IPv4 is being phased out as there are no longer enough possible combinations to provide a unique identifier for every one. IPv6 was introduced, containing 128 bits where IPv4 only has 32. This means it has 1028 times more possible combinations, sufficient for the increased amount of devices connected to the internet for now. Every device has a private IP address which is assigned by the router. Then the router networks each device with the public internet, all sharing the same public IP given by the ISP (Internet Service Provider) to the router.

While IP addresses provide a great way to store the location of a device in a digital landscape, this also means that the services your internet communicates with will also have access to your digital location, which also gives a rough real world location of where the physical system is. IP addresses can be protected through VPNs and similar products, but they sacrifice a substantial amount of speed for one layer of security, making the tradeoff not worth it for most people.

IPv4 Example: 192.16.1.127
IPv6 Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
